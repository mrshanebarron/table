# Laravel Design Table

Data table component with sorting for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/table
```

## Usage

### Livewire Component

```blade
<livewire:ld-table />
```

### Blade Component

```blade
<x-ld-table />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-table-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-table-views
```
